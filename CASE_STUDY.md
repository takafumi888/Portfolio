# CASE STUDY: Shopify Portfolio Works

このCASE_STUDY.mdでは、Shopifyエンジニア転職用ポートフォリオに掲載する3つのShopifyデモストアについて、制作意図、担当範囲、Shopify設計、実装内容、学びを整理します。

## Purpose

このポートフォリオは、単に画面を作るだけではなく、Shopify制作会社の実務を意識して以下を示すことを目的としています。

- Shopifyテーマの構造理解
- 商品・コレクション設計
- メタフィールド活用
- Journal / FAQなどのコンテンツ設計
- 商品CSVを使った商品管理理解
- 実装レポートによるドキュメント化
- PC/SP対応
- 採用担当者が確認しやすい情報整理

## Work 01: DAILY石鹸

### Project Type

Shopifyスクール課題

### Demo / Password

- Password：`portfolio`

### Summary

DAILY石鹸は、石鹸ブランドを想定したShopifyデモストアです。清潔感、信頼感、商品の特徴の伝わりやすさを重視して制作しました。

### Role

- 商品登録
- コレクション設計
- 基本ページ構成
- レスポンシブ確認

### Shopify Structure

- 商品情報が伝わりやすい商品詳細構成
- トップページからコレクションへの導線
- 商品一覧から購入検討に進みやすい導線
- 清潔感を意識したブランド表現

### Implementation

- テーマの基本構造を確認しながら必要箇所を調整
- 商品登録とコレクション設計を実施
- PC/SPで画像、余白、テキスト量を確認
- 商品詳細ページの見やすさを重視

### Learnings / Improvements

Shopifyテーマの基本構造、商品とコレクションの関係、トップページから商品詳細までの導線設計を学びました。今後はFAQ、レビュー導線、メタフィールドによる商品情報拡張を追加したいです。

## Work 03: sereine

### Project Type

Shopifyスクール卒業課題

### Demo / Password

- Demo URL：https://fleur-sereine-graduate.myshopify.com/
- Password：`sereine`

### Summary

sereineは、Shopify公式テーマRiseをベースに制作した卒業課題です。商品設計、コレクション設計、メタフィールド、Journal、FAQ、商品CSV、実装レポートまで含め、実務に近い制作フローを意識しました。

### Role

- Riseテーマカスタマイズ
- 商品設計
- コレクション設計
- メタフィールド設計
- Journal作成
- FAQ作成
- 商品CSV作成
- 実装レポート作成
- レスポンシブ確認

### Shopify Structure

#### Theme

- Shopify公式テーマ Rise

#### Product / Collection

- 商品情報、商品説明、価格、バリエーション、商品画像を整理
- 用途やカテゴリに応じてコレクションを設計
- トップページから商品一覧・商品詳細へ進みやすい導線を確認

#### Metafields

- 商品補足情報
- 成分・素材
- おすすめポイント
- 管理画面で更新しやすい表示項目

#### Journal / FAQ

- ブランド理解を深める記事導線
- 商品選びの参考になるコンテンツ
- 配送・返品・商品仕様に関する購入前不安の解消

#### Product CSV

- 商品登録の効率化
- 商品情報の整理
- 運用を想定した商品管理

### Implementation

- Riseテーマの構造を把握し、必要な箇所を調整
- 商品・コレクション・コンテンツの関係を整理
- 管理画面で更新しやすい設計を意識
- PC/SPで読みやすいレイアウトに調整
- 実装レポートとして制作内容を記録

### Learnings / Improvements

Shopify公式テーマをベースにした制作フロー、商品設計とテーマ表示の関係、メタフィールドを使った情報拡張、Journal / FAQによるECサイトの情報設計、商品CSVを使った運用視点を学びました。

今後はLiquidカスタムセクションの追加、商品レコメンドの強化、レビューアプリ表示の検証、Shopify Flowを使った運用自動化、パフォーマンス改善、アクセシビリティ改善に取り組みたいです。

## Overall Learning

今回の制作を通じて、Shopifyエンジニアに必要なスキルは単なるコーディングだけではないと理解しました。クライアントが商品を登録・更新しやすい設計、商品情報をどのページでどう見せるか、購入前の不安をFAQやJournalでどう解消するか、実装内容を説明できるようにドキュメント化することが重要だと感じています。
