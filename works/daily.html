Project Summary

概要

Shopifyスクール課題として制作した、石鹸ブランドのデモストアです。購入前に商品の特徴が伝わり、清潔感のあるブランドとして認識できる構成を意識しました。

Scope

担当範囲

  • テーマカスタマイズ
  • 商品登録
  • コレクション設計
  • 基本ページ構成
  • レスポンシブ確認

Shopify Structure

Shopify設計

商品情報の見やすさとコレクションへの導線を優先しました。商品詳細では特徴・価格・購入導線が迷わず確認できるよう、基本的なShopifyストア構造の理解を目的に制作しています。

Implementation

実装内容

  • ブランドの清潔感を意識した余白と配色調整
  • 商品一覧から商品詳細へ進みやすい導線整理
  • PC/SPでカードと画像が崩れない表示確認
  • トップページ、商品一覧、商品詳細の基本構成確認

Documentation

ドキュメント化

制作意図、担当範囲、学び、今後の改善点をCASE_STUDYに整理し、採用担当者が制作背景を追える状態にしています。

Challenges & Improvements

課題と改善

今後はFAQ、商品レビュー導線、メタフィールドによる成分情報の拡張を追加し、ECサイトとしての購入前不安をより減らせる構成に改善したいです。
