# Shopify Theme Developer Portfolio

Shopifyエンジニア転職を目的に制作したポートフォリオサイトです。

GitHub Pagesで公開できる静的HTML / CSS / JavaScript構成で制作しています。
Shopifyテーマ実装の学習に加えて、商品設計、コレクション設計、メタフィールド、Journal、FAQ、商品CSV、実装レポートなど、ECサイト制作・運用を想定した要素を整理して掲載しています。

## Demo

```text
https://takafumi888.github.io/Portfolio/
```

## Concept

**Shopifyテーマ実装・ストア設計・運用理解を伝える、実務を意識したECポートフォリオ**

未経験からShopifyエンジニアを目指すにあたり、単なるWeb制作作品集ではなく、Shopify制作会社・EC支援会社の現場を意識したポートフォリオとして制作しました。

* 各Works詳細ページに概要、取り組んだこと、Shopify設計、実装内容、パスワードを掲載

## Development

静的サイトのため、ブラウザで `index.html` を直接開いて確認できます。
ローカルサーバーで確認する場合は、以下のコマンドを使用します。

```bash
python3 -m http.server 8000
```

ブラウザで以下にアクセスします。

```text
http://localhost:8000
```

## Goal

Shopify制作会社・EC支援会社・Web制作会社の採用担当者に対して、以下を伝えることを目的としています。

* Shopifyテーマ実装の基礎を学習している
* 商品・コレクション・メタフィールドなど、ストア設計を意識して制作している
* FAQ、Journal、商品CSVなど、ECサイト運用を想定した要素を制作・整理している
* 実装内容をREADMEやCASE_STUDYとしてドキュメント化している
* 今後もLiquid、テーマカスタマイズ、運用改善の理解を深めていく姿勢がある
